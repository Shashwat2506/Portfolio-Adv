# ⚙ Installation


Clone Repository


```bash
git clone https://github.com/Shashwat2506/Portfolio-Adv.git
```


Enter Project


```bash
cd Portfolio-Adv
```


Install Packages


```bash
npm install
```


Run Development Server


```bash
npm run dev
```


Open:


```
http://localhost:3000
```
